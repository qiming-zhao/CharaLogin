# CharaLogin

一个现代化的登录界面，包含动画效果和交互功能。

![image-20260328181903647](assets\image-20260328181903647.png)

## 功能特性

- 响应式设计，支持桌面和移动端
- 密码可见性切换
- 记住我功能
- 登录表单验证
- 左侧装饰动画（四个表情呆萌的小人以及一定的交互动画效果）
- 现代简洁的UI设计

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript

## 使用方法

直接在浏览器中打开 `index.html` 即可预览。

```bash
# 或者使用简单的HTTP服务器
npx serve .
# 或
python -m http.server 8000
```

## 文件结构

```
├── index.html          # 主页面
├── style.css          # 自定义样式
├── script.js          # 交互逻辑
└── tailwind_output.css # Tailwind CSS输出
```
